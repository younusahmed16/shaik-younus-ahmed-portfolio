# Younus Ahmed Shaik - Personal Portfolio

A premium, modern, and highly professional personal portfolio website designed for a Software Developer. Built with React, Tailwind CSS, and Framer Motion, it features a clean, elegant, and futuristic UI with a dark theme and glassmorphism effects.

## ✨ Features

- **Modern UI/UX:** Clean, dark theme with gradient accents and glassmorphism effects.
- **Fully Responsive:** Optimized for mobile, tablet, and desktop devices.
- **Smooth Animations:** Integrated with Framer Motion for scroll reveal animations, typed text effects, and micro-interactions.
- **Dynamic Content:** Easily update your skills, projects, and education in a single `data.ts` file.
- **Interactive Sections:**
  - **Hero:** Animated introduction and quick links.
  - **About:** Professional summary and strengths.
  - **Skills:** Animated progress bars categorizing technical skills.
  - **Projects:** Interactive cards showcasing recent work.
  - **Education & Certifications:** Timeline layout and achievement highlights.
  - **Contact:** Functional-looking contact form and social links.

## 🛠 Tech Stack

- **Frontend Framework:** [React 19](https://react.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://motion.dev/)
- **Icons:** [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You need to have Node.js installed on your machine.
- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/younusahmed/your-repo-name.git
   ```
2. Navigate into the project directory
   ```sh
   cd your-repo-name
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start the development server
   ```sh
   npm run dev
   ```

## 📝 Customization

All the personal information, skills, projects, and education details are centralized in the `src/data.ts` file. 
Open this file to update the content with your own details without needing to dig through the component code.
